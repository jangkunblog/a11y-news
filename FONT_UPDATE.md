# 🎨 폰트 및 헤딩 스타일 업데이트

## ✅ 적용된 변경사항

### 1. IBM Plex Sans KR 폰트 적용
- Google Fonts에서 IBM Plex Sans KR 로드
- 전체 사이트에 일관되게 적용
- 한글 가독성 향상

### 2. 접근성 아이콘 추가 (헤딩 레벨별)
- **h2** (##): ♿ - 유니버설 디자인/메인 접근성
- **h3** (###): 👁️ - 시각 접근성
- **h4** (####): ✋ - 키보드/터치 접근성
- **h5** (#####): ✓ - 준수사항/체크리스트
- **h6** (######): ★ - 중요 포인트

### 3. 구조적 들여쓰기
- **h2**: 패딩 없음, 상단 구분선
- **h3**: 1.5rem 들여쓰기
- **h4**: 3rem 들여쓰기
- **h5**: 4.5rem 들여쓰기
- **h6**: 6rem 들여쓰기

### 4. 콘텐츠 자동 들여쓰기
- 각 헤딩 다음의 단락, 리스트 등이 해당 레벨만큼 들여쓰기됨
- 계층 구조가 시각적으로 명확하게 표현됨

### 5. 레벨별 상단 패딩
- **h2**: 3rem (가장 큰 간격)
- **h3**: 2.5rem
- **h4**: 2rem
- **h5**: 1.5rem
- **h6**: 1.2rem

### 6. 반응형 디자인
- 모바일에서는 들여쓰기 간격이 자동으로 줄어듦
- 720px 이하에서 최적화된 간격 적용

---

## 🎯 효과

1. **가독성 향상**: IBM Plex Sans KR의 우수한 한글 렌더링
2. **구조 명확화**: 시각적 계층이 명확하게 구분됨
3. **접근성 강조**: 아이콘으로 각 섹션의 주제를 직관적으로 파악
4. **전문성**: 체계적인 문서 구조로 전문성 있는 느낌
